## How to Take Screenshots

### macOS
- Full screen: `Cmd + Shift + 3`
- Selected area: `Cmd + Shift + 4`
- Window: `Cmd + Shift + 4`, then press `Space`, then click window

### Windows
- Full screen: `Win + Print Screen`
- Selected area: `Win + Shift + S`

### Browser DevTools (Recommended)
1. Open DevTools (F12)
2. Click the device toggle (Cmd+Shift+M / Ctrl+Shift+M)
3. Set viewport to 1920x1080
4. Take screenshot using browser's built-in tool

## After Taking Screenshots

1. Save them in this directory with the exact names above
2. Run: `git add screenshots/*.png`
3. Run: `git commit -m "docs: Add screenshots to README"`
4. Run: `git push origin main`

The screenshots will automatically appear in the README!
